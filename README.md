# Bitasmbl-Responsive-Developer-Portfolio-Landing-Page-02a8c1-b9bbb1a779c7

## Description
A single-page responsive portfolio landing site for a junior developer featuring hero, about, projects, and contact sections using semantic HTML and either plain CSS or Tailwind CSS.

## Tech Stack
- Bootstrap
- Tailwind CSS

## Requirements
- Implement a mobile-first responsive layout that reflows sections for small screens.
- Create three project cards with titles, descriptions, and GitHub links.
- Add basic hover animation to CTA button and project cards.
- Use Tailwind CSS via CDN instead of custom CSS.
- Deploy the landing page using GitHub Pages or Vercel.

## Installation
bash
git clone https://github.com/he1snber8/Bitasmbl-Responsive-Developer-Portfolio-Landing-Page-02a8c1-b9bbb1a779c7.git
cd Bitasmbl-Responsive-Developer-Portfolio-Landing-Page-02a8c1-b9bbb1a779c7


## Usage
Open `index.html` in a browser.

## Implementation Steps
1. Create semantic HTML structure with hero, about, projects, and contact sections.
2. Include Tailwind CSS via CDN in `index.html`.
3. Build a mobile-first layout using Tailwind utility classes.
4. Add three project cards with titles, descriptions, and GitHub links.
5. Implement hover animations for the CTA button and project cards using Tailwind classes.
6. Deploy `index.html` using GitHub Pages or Vercel.
